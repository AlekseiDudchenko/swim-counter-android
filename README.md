# Swim Counter (Android)

Minimal Android app for swim training sessions.

## Features (MVP)
- Distance counter (+50m / -50m)
- Set counter (+1 / -1)
- Reset session
- Jetpack Compose
- No network, no persistence

## Tech stack
- Kotlin
- Jetpack Compose
- MVVM (lightweight)
- StateFlow
